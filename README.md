# 32-bit RISC-V Mini CPU (Verilog)

This repository contains the RTL design and simulation of a basic 32-bit RISC-V mini processor using Verilog HDL.

## 🧱 Features
- ALU supporting basic arithmetic and logic operations
- Control Unit for opcode decoding
- Register File with 32 registers
- Program Counter and Instruction Memory
- IF/ID pipeline register for staging
- CPU Top module integration
- Testbench for simulation
- Waveform analysis using GTKWave

## 📦 Tools
- Icarus Verilog (iverilog)
- GTKWave
- VS Code
